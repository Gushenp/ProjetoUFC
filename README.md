🥊 Projeto UFC - Simulador de Lutas em Java
Este é um projeto simples em Java que simula lutas entre lutadores,
inspirado no estilo do UFC. O sistema permite cadastrar lutadores,
definir categorias com base no peso e realizar lutas com resultados
aleatórios.
---
📌 Funcionalidades
Cadastro de lutadores\
Definição automática de categoria por peso\
Simulação de lutas\
Atualização de histórico (vitórias, derrotas e empates)\
Validação de luta (mesma categoria e lutadores diferentes)
---
🧱 Estrutura do Projeto
📁 Lutadores.java
Representa um lutador com atributos como nome, idade, peso e cartel.
📁 Luta.java
Responsável por validar e executar lutas entre lutadores.
📁 ProjetoUFC.java
Classe principal com menu interativo no terminal.
---
▶️ Como Executar
``` bash
git clone https://github.com/seu-usuario/projeto-ufc.git
cd projeto-ufc
javac projetoufc/*.java
java projetoufc.ProjetoUFC
```
